# LinkedIn Post Creation Skill

A comprehensive skill for creating high-engagement B2B LinkedIn posts using research-driven, hook-focused methodology.

## Overview

This skill guides you through a 9-phase workflow for creating LinkedIn posts that stop the scroll and drive engagement. Built on proven frameworks from top LinkedIn creators including Erica Schneider, Matt Swain, and Anthony Pierri (Fletch).

## Key Features

- **Hook-Obsessed Approach**: 40% of time spent on hooks using TAS, PAS, and SCQA frameworks
- **Mobile-First Formatting**: One sentence per line, spaces between, NO EMOJIS for B2B
- **Research-Driven**: Integrates Exa.ai and Firecrawl MCP tools for deep research
- **Multiple Input Types**: Topic, brain dump, similar post reverse engineering, blog/article conversion
- **Strong POV Required**: Vanilla content fails - contrarian angles win
- **11-Point Editing Framework**: Adapted specifically for LinkedIn's format

## The 9-Phase Workflow

### Phase 0: Strategic Planning & Research
- Input analysis (4 types supported)
- Research strategy using web_search_exa (primary) and firecrawl (fallback)
- Content format selection from 10 proven formats

### Phase 1: Seed Content Development
- Collect raw material from research
- Extract golden nuggets worth developing
- Build content foundation

### Phase 2: Reverse Outline
- Create structured outline from seed content
- Organize ideas logically
- Plan post flow

### Phase 3: Y+U+I Analysis
- **Why**: Big picture context
- **For You**: Audience perspective
- **For Me**: Your unique angle

### Phase 4: Parts Analysis
- Face Value, Truth, Assumption
- Consequences, Story
- Zoom In, Zoom Out

### Phase 5: Hook Creation (MOST CRITICAL)
⚠️ SPEND 40% OF YOUR TIME HERE ⚠️

**The 3 Core Frameworks:**
1. **TAS** (Thesis, Antithesis, Synthesis)
2. **PAS** (Problem, Agitate, Solution)
3. **SCQA** (Situation, Complication, Question, Answer)

**The 4 Foundational Hook Principles:**
1. Poke at the Pain
2. Add Credibility
3. Get Specific with Outcomes
4. Leave a Cliffhanger

### Phase 6: Post Creation
- Body development using chosen framework
- Strong POV throughout
- Natural flow from hook

### Phase 7: Mobile Formatting
**CRITICAL RULES:**
- One sentence per line
- Blank lines between sentences
- NO paragraphs
- Short sentences (10-15 words)
- NO EMOJIS (B2B professional)

### Phase 8: 11-Point Editing
1. Redundancy elimination
2. Active voice conversion
3. Parallelism
4. Tense consistency
5. Sentence structure optimization
6. Specificity enhancement
7. Argument amplification
8. Structural strategy
9. Takeaway strengthening
10. Grammar & spelling
11. Final polish

## Hook Quality Checklist

Before finalizing any post, verify your hook:
- [ ] Specific (not generic)
- [ ] Contrarian or unexpected
- [ ] Emotional (creates feeling)
- [ ] Relatable (audience sees themselves)
- [ ] Data-driven (when relevant)
- [ ] Cliffhanger (opens a loop)
- [ ] Mobile-friendly (scans well)
- [ ] Authentic (sounds like you)

## Mobile Formatting Rules

### Visual Spacing
```
✅ CORRECT:
Hook sentence that grabs attention.

Second line adds context.

Third line creates tension.

❌ WRONG:
Hook sentence that grabs attention. Second line adds context. Third line creates tension.
```

### Sentence Length
- Aim for 10-15 words per sentence
- Break longer sentences into shorter ones
- Make every word count

### No Emojis
- B2B professional context
- Emojis reduce credibility
- Text-only for impact

## Common Mistakes to Avoid

1. **Generic Hooks**: "Here's what I learned..." → Specific outcome instead
2. **Paragraph Formatting**: Multiple sentences together → One per line
3. **Safe Takes**: Vanilla opinions → Contrarian angles
4. **Skipping Hook Time**: 10% on hooks → 40% minimum
5. **Adding Emojis**: 👍 → Professional text only
6. **Weak Cliffhangers**: Full reveal → Create tension
7. **Passive Voice**: "It was done" → "We did it"
8. **Long Sentences**: 25+ words → 10-15 words max
