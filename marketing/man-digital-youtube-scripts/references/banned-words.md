# Banned Words & Phrases (absolute)

Scan the finished script against every line below before delivering it. These are banned
regardless of context — this list is verbatim from Romeo's blocklist (2026-07-22). If a
synonym is doing the same job as one of these (vague transformation, storytelling metaphor,
empty superlative), treat it as banned too, even when it isn't spelled out below.

## Landscape / era framing

- in the world of
- in the landscape of
- in the era of
- realm
- ever-evolving
- evolving

## Vague transformation verbs

- elevate / elevated
- discover
- supercharge
- unlock
- unleash
- solidify
- enhance / enhancing / enhanced
- thrive

- tailored

- game changer
- pivotal
- captivating
- dazzle
- power / profit-power / revenue-driven / data-driven

## Handling an unavoidable literal use

If a banned word is unavoidable in a strictly literal, technical sense (for example, a
HubSpot feature actually named with one of these words), do not silently swap in a euphemism
that changes the meaning — flag it to Romeo and ask before including it, rather than hiding
the conflict.
