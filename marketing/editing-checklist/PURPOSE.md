# Why editing-checklist Lives Here (Not in blog-production)

This is a **standalone editorial quality framework** — not a pipeline step.

Run it on:

- Blog posts (after step 06 Write)
- LinkedIn posts
- Email copy
- Landing pages
- Anything written

## What's Inside

Eight reference docs covering:

- **brevity-masterclass** — cutting flab
- **cliches-masterclass** — what to avoid + better alternatives
- **readability-masterclass** — sentence rhythm, paragraph length, scannability
- **confidence-masterclass** — strong, definitive writing
- **paragraph-structure-guide** — patterns that work
- **defensive-writing-guide** — what makes writing weak (hedges, qualifiers, throat-clearing)
- **advanced-editing-guide** — high-level structural edits
- **quick-reference-lists** — fast lookups during editing

Plus the main `SKILL.md` (~67KB) that ties it all together.

## Use It

In Claude Code:

> "Apply the editing-checklist skill to this draft."

Or invoke directly:

> "Use editing-checklist references/brevity-masterclass and references/cliches-masterclass to edit this section."

See the main `README.md` and `SKILL.md` inside this folder for details.
