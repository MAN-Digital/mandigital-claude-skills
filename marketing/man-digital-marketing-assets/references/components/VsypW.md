Current live lookup note: this node was not found in the active Marketing `Playground.pen` path during the 2026-05-28 Pencil MCP check, so treat it as a registry/library reference and verify availability in the active Pencil file before copying.

## Role

Use as a soft background texture when a light graphic needs depth without adding another content system.

## Visual Structure

- Frame about `698 x 703`.
- Three subtle concentric or overlapping circles.
- Low-contrast gray/blue-gray treatment.
- Intended for layer 0 behind content.

## When To Use

- Light-background explainers with large negative space.
- Carousel-derived or social crops where a soft background needs more texture.
- As an alternative to repeatedly inventing pale ellipses.

## Editing Notes

Keep opacity low, usually around `20-35%` depending on background. Do not place it over text, icons, or connector lanes. If the node is unavailable in the active Pencil file, recreate it as editable low-opacity ellipses only when the composition actually needs soft background depth.

## Reject When

- The graphic already has dot grids, radar arcs, screenshot frames, or dense connector systems.
- The circle would sit behind a card edge and create accidental halos.
- The decoration exists only because the frame feels empty; improve the layout first.
