# ujAy2 - Four-Layer Blueprint Group

Source parent: `llyux`

## Role

Use as the complete four-layer operating model: Inputs, Processing, Activation, Measure + Learn.

## Visual Structure

- Four large pale row backgrounds.
- White content panels inside each row.
- Numbered left rail and vertical layer labels.
- Capability tiles arranged across each row.
- Connector lines linking rows.

## Editing Notes

Clone the whole group for a stable blueprint. If a layer count changes, adjust row backgrounds, vertical labels, capability spacing, and connectors together.
