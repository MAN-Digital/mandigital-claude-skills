# lLDFt - Pre-Sales Across Disconnected Tools

Source: `/Users/romeoman/Documents/Marketing/Design/Pencil/Playground.pen`

Observed frame: `Prospecting-Agent - 1080x1350 - c4`, 1080 x 1350.

## Role

Use to explain what happens when pre-sales work is spread across multiple disconnected tools.

## Visual Structure

- Light Ghost White background with large pale circle.
- Title: `What happens when pre-sales runs across tools`.
- Subtitle: `Each step works. The context gets weaker`.
- Four large circle nodes: Signal, Data, Research, Outreach.
- Bottom circle node: CRM sync-back.
- Thin blue connectors show exports, enrichment, and sync-back.
- Red accent arrows call out strong individual tools and weaker context across the motion.
- Bottom thesis: disconnected tools create handoffs, not a connected pre-sales system.

## Use When

Use for data leakage, handoff friction, disconnected RevOps systems, or "tools are not a system" arguments.

## Editing Notes

Keep the connector labels short. The red accents should remain warning/cost signals, not general decoration.
