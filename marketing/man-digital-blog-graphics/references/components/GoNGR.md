# GoNGR - Thin Outline Icon Style Reference

Source: user-provided Pencil/registry icon example. Current live lookup note: this node was not found in the active Marketing `Playground.pen` path during the 2026-05-28 Pencil MCP check.

## Role

Use as a style reference for thin, minimal, Streamline-like icons.

## Visual Structure

- Clean line icon treatment.
- Should feel lighter than bold filled hero icons.
- Use Medium Blue `#000FC4` unless the surrounding system requires a semantic warning color.

## Editing Notes

When the exact node is unavailable, choose an existing Playground Streamline-style icon with similar stroke weight and semantics. Do not create heavy icon-font blobs or oversized filled icons as a replacement.
