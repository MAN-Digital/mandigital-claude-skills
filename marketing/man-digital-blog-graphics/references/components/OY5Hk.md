# OY5Hk - Thin Outline Icon Style Reference

Source: user-provided Pencil/registry icon example. Current live lookup note: this node was not found in the active Marketing `Playground.pen` path during the 2026-05-28 Pencil MCP check.

## Role

Use as a style reference for thin utility icons in comparison, routing, governance, and data-model graphics.

## Visual Structure

- Minimal stroke icon.
- Works best with 32 px icon slots and light blue/purple surfaces.
- Supports labels without becoming the focal point.

## Editing Notes

Use one icon language per graphic. If one card uses this thin style, all sibling cards should use the same visual weight.
