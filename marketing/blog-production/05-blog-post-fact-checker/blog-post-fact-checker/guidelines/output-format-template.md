This template defines the exact structure and format for all fact-check reports generated by this skill.

- Primary: `mcp__MCP_DOCKER__web_search_exa`
- Supporting: Manual source evaluation

**Date Preference:**
- Target: 2024-2025 sources for current claims
- Acceptable: 2023 for established facts
- Flag: Pre-2023 data presented as current

**Source Authority:**
- Preferred: Tier 1 sources (research firms, academic, major publications)
- Acceptable: Tier 2 sources (industry publications, known SaaS blogs)
- Avoided: Tier 3 sources (unknown blogs, content farms)

**Cross-Reference Threshold:**
- Critical claims: 2+ authoritative sources
- High priority: 1-2 authoritative sources
- Medium priority: 1+ credible source

## Usage Notes

### When Generating Reports:

1. **Always include both sections:**
   - Detailed verification for EACH claim
   - Summary statistics

2. **Be explicit in decisions:**
   - Clear PASS/FAIL/MINOR ISSUES designation
   - Specific actions required
   - Rationale for decision

3. **Provide actionable recommendations:**
   - Not just "unsupported" but "Update to [corrected text]"
   - Include suggested citations
   - Offer alternative phrasings

4. **Display as artifact:**
   - Save to disk first
   - Then display same content as artifact
   - User can review without conversation clutter

### Quality Checklist Before Sending:

- [ ] All claims have verification status
- [ ] Each claim has Exa query documented
- [ ] Sources include authority tier
- [ ] Publication dates noted
- [ ] Clear PASS/FAIL decision
- [ ] Specific recommendations provided
- [ ] Summary statistics completed
- [ ] Next steps clearly stated
- [ ] Saved to disk AND displayed as artifact

## Examples of Good vs. Bad Reporting

### ❌ Bad Example:
```
Claim: "85% of marketers use AI"
Status: Not verified
```

### ✅ Good Example:
```
Claim: "85% of marketers use AI tools"
Status: ⚠️ PARTIALLY VERIFIED
Confidence: Medium

Verification:
Exa Query: "AI adoption marketing statistics 2024"
Sources Found:
1. HubSpot State of Marketing 2024 - Tier 1 - Shows 82% adoption
2. Salesforce Marketing Trends 2024 - Tier 1 - Shows 87% adoption

Assessment: The claim of "85%" falls within the range reported by two authoritative Tier 1 sources (82-87%). The claim is substantially accurate but should be updated with the specific source citation.

Recommendation: Update to: "According to HubSpot's State of Marketing 2024 report, 82% of marketers now use AI tools in their workflows."
```
